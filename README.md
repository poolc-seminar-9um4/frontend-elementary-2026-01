# Front-end Elementary Seminar 2026-01

## 폴더 구조 설명

### materials
세미나 보조 자료를 정리해 둔 폴더입니다.

### repository
세미나 진행시 살펴본 코드들을 정리해 둔 폴더입니다.

### day_02 (=main)
1~2회차 세션에서 다룬 내용들이 포함되어 있는 Tag입니다.
