CSS 속성 종합 데모

background, font, spacing, layout, position 속성을 한 화면에서 확인

1) 폰트 + 크기/간격 + 테두리

CARD A

Typography

color, font-size, font-weight, font-family, text-align, line-height를 적용한 텍스트입니다. 카드의 width, height, padding, margin, border도 함께 사용합니다.

CARD B

Spacing

내부는 padding, 카드 사이 간격은 flex의 gap, 전체 여백은 section의 margin과 padding으로 표현됩니다. 실무에서는 이런 값들을 일관되게 설계합니다.

CARD C

Border

border 속성으로 요소 경계를 만들고, border-radius를 더하면 부드러운 카드 UI를 만들 수 있습니다. 배경색 대비를 주면 구조가 더 잘 보입니다.
