이 문서는 style 태그나 외부 CSS 파일 없이, HTML 태그 속성만 사용해서 꾸몄습니다. 배경색, 글자색, 정렬, 간격을 전부 각 요소에 직접 적어야 합니다.

왜 CSS가 분리되었을까?
  • 디자인 속성이 HTML 구조와 섞여서 코드가 길어집니다.
  • 같은 스타일을 여러 태그에 반복 작성해야 합니다.
  • 색상/폰트 변경 시 모든 태그를 하나씩 수정해야 합니다.
  • 반응형 레이아웃, 애니메이션 같은 현대 기능을 쓰기 어렵습니다.
결론: HTML은 구조, CSS는 표현을 맡기면 재사용성과 유지보수성이 크게 좋아집니다.
